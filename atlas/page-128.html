GEO-INFER
Open source geospatial methods for Active Inference & Beyond

https://github.com/ActiveInferenceInstitute/GEO-INFER

 GEO-INFER is an open source geospatial active inference project developed by the Active Inference Institute
to apply probabilistic generative modeling to spatial and temporal data. It is organized as a modular ecosystem,
with conceptual components for math and inference (e.g., Bayesian modeling, active inference, agents),
geospatial-temporal infrastructure (space, time, IoT, data), domain modules (such as agriculture, health,
economics, risk, and logistics), operations and security (including data operations and a GEO‑INFER‑SEC security
and privacy framework), and higher-level application, simulation, and governance layers.

Within this ecosystem, modules like GEO‑INFER‑APP focus on user-facing applications, GEO‑INFER‑SIM on
simulation, GEO‑INFER‑ANT on complex systems, and GEO‑INFER‑CIV/ORG/COMMS on civic, organizational, and
communication aspects, all linked via an explicit dependency graph and a complete module dependency matrix
that document how capabilities build on one another. Community calls and open working meetings hosted by the
Active Inference Institute support collaborative development, helping contributors learn geospatial methods,
bioregional modeling, and nested active inference systems while extending the GEO‑INFER module set over time.
