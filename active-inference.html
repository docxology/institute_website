Active Inference

A framework for understanding perception, action, learning, cognition, and adaptive systems.

The source PDF introduces Active Inference as the central scientific and practical framework around which the Institute organizes education, research, applications, and ecosystem support.

Source coverage: Pages 15-21 and 181-185. See the page-by-page source atlas.

What Active Inference Provides

Active Inference connects perception, action, uncertainty, and model-based adaptation. The Institute's public material presents it as a shared framework for scientific inquiry, engineering practice, education, and collective sensemaking.

How to Approach the Field

The PDF directs newcomers to the living document, the Start page, livestreams, podcasts, textbook groups, readings, and implementations. This site separates those entry points into stable public pages while keeping source-page references visible.

Implementations

Implementation pathways include RxInfer.jl, PyMDP, SPM, and symbolic cognitive robotics. These tools support modeling, simulation, inference, and applied experimentation.

Applications

Active Inference appears across domains including biology, neuroscience, mental health, robotics, education, economics, physics, social systems, logistics, DeSci, and scientific method.
